¿Cómo registrar un contrato?

  • 1
    Registro del contrato:
    Para registrar un contrato, el empleador debe estar inscrito en el sistema de registro para empleadores. Si aún no está registrado, puede hacerlo a través del sitio web de trámites de la Dirección del Trabajo. Para conocer el paso a paso del proceso, haga clic en el enlace "¿Cómo registrarlo?" o vea el siguiente video.
